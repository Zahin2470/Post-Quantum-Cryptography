## 📌 Overview

This project explores **Post-Quantum Cryptography (PQC)** through the lens of **network traffic analysis and anomaly detection**. As quantum computers pose an increasing threat to classical cryptographic schemes, this work investigates how PQC protocols manifest in network traffic and how machine learning techniques can be applied to analyze and detect anomalies within such encrypted communications.

## 🧠 Key Topics

- **Post-Quantum Cryptographic (PQC) Algorithms** — CRYSTALS-Kyber, CRYSTALS-Dilithium, FALCON, SPHINCS+
- **Network Traffic Analysis** — Feature extraction, protocol fingerprinting
- **Anomaly Detection** — ML-based detection of unusual traffic patterns in PQC-encrypted communications
- **Machine Learning Models** — Classification, clustering, and anomaly detection pipelines
